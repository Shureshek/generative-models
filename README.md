# Generative Models Portfolio
Этот репозиторий содержит коллекцию проектов по генеративным моделям, реализованных с использованием современных подходов глубокого обучения.

## Доступные проекты

### 1. **[WGAN-GP: Wasserstein GAN with Gradient Penalty](./WGAN-GP/)**
**Технологии:** PyTorch, CNN, GAN

**Описание:** Реализация усовершенствованной генеративно-состязательной сети для стабильного обучения и генерации лиц аниме-персонажей. Проект решает классические проблемы GAN через применение Wasserstein distance и градиентного штрафа.

**Ключевые особенности:**
- Решение проблем vanishing gradients и mode collapse
- Gradient Penalty для обеспечения 1-Липшицевости
- Полный пайплайн: от preprocessing до анализа результатов
- Генерация качественных изображений 64×64

**Ссылки:** [WGAN-GP](./WGAN)
