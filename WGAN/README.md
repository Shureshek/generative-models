# WGAN-GP for Anime Face Generation
Реализация продвинутой генеративно-состязательной сети (WGAN-GP) для создания высококачественных изображений лиц аниме-персонажей. Проект демонстрирует решение классических проблем GAN через применение Wasserstein distance и градиентного штрафа.

### Ключевые особенности

- **Стабильное обучение**: WGAN-GP решает проблемы vanishing gradients и mode collapse
- **Качественная генерация**: Создание разнообразных и четких изображений 64x64
- **Профессиональный пайплайн**: Полный цикл от preprocessing до анализа результатов
- **Интерактивная визуализация**: Мониторинг обучения в реальном времени

## Архитектура

### Модели

| Компонент | Архитектура | Назначение |
|-----------|-------------|------------|
| **Генератор** | ConvTranspose2d (5 слоёв) | Преобразует шум в изображения |
| **Критик** | Conv2d (5 слоёв) | Оценивает "реальность" изображений |

### Ключевые инновации

1. **Wasserstein Loss**: Заменяет бинарную кросс-энтропию на Wasserstein distance
2. **Gradient Penalty**: Регуляризация для обеспечения Липшицевости критика
3. **n-critic training**: Балансировка обучения генератора и критика
